L.esri.Vector.Basemap

L.esri.Vector.Basemap is used to display Esri hosted vector basemaps and attributes data providers appropriately. The Terms of Use for Esri hosted services apply to all Leaflet applications.

Inherits from L.Layer

Constructor

Constructor Description
L.esri.Vector.basemap(<String> key, , <Object> options) key refers to the specific basemap you'd like to add.
Vector Basemaps

Worldwide coverage at a variety of zoom levels.

  • Newspaper (added in 1.0.2)
  • MidCentury (added in 1.0.2)
  • Spring (added in 1.0.2)
  • Gray
  • DarkGray
  • Topographic
  • Navigation
  • Streets
  • StreetsNight
  • StreetsRelief

Options

L.esri.Vector.Basemap accepts all L.Layer options.

Methods

L.esri.Vector.Basemap inherits all methods from L.Layer.

Example

<script src="./esri-leaflet-vector.js"></script>
var map = L.map('map').setView([37.75,-122.45], 12);
L.esri.Vector.basemap("Newspaper").addTo(map);
